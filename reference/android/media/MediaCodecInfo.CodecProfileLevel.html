level

added in API level 16 
int level

Defined in the OpenMAX IL specs, depending on the type of media this can be OMX_VIDEO_AVCLEVELTYPE, OMX_VIDEO_H263LEVELTYPE OMX_VIDEO_MPEG4LEVELTYPE, OMX_VIDEO_VP8LEVELTYPE or OMX_VIDEO_VP9LEVELTYPE. Note that VP9 decoder on platforms before N may not advertise a profile level support. For those VP9 decoders, please use MediaCodecInfo.VideoCapabilities to determine the codec capabilities.
