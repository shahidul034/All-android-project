CarrierConfigManager

public class CarrierConfigManager
extends Object

java.lang.Object
   ↳ android.telephony.CarrierConfigManager

Provides access to telephony configuration values that are carrier-specific.

Instances of this class must be obtained using Context.getSystemService(Class) with the argument CarrierConfigManager.class or Context.getSystemService(String) with the argument Context.CARRIER_CONFIG_SERVICE.
